Detta är ett projekt gjort på några dagar för att skapa ett backend och frontend. Backend med C# och .NET. Frontend med Angular. Här används JSON Web Token (JWT) för att sköta och kolla inloggning är korrekt mot backend. Det används ingen databas utan allt är serverless så inget sparas permanent när backend är igång. Det körs endast lokalt. Bootstrap används för responstivitet och skapa en snygg och enhetlig design. Uppgiften gick ut på att skapa en sida för att skapa böcker. Där man kan skapa, redigera och ta bort böcker. Sen även ha en sida med sina favoritcitat. https://github.com/a19jimsa/angular-challange
Detta är ett projekt med React. För backend används Node.js med Express och MongoDB som databas.

Mitt första spelprojekt 2013 i C++
